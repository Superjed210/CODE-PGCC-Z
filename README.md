# CODE Club Website
Featuring a PGCC Interactive Campus Map

This is the official website for CODE (Community of Driven Engineers). It showcases who we are, what we do, and includes an interactive, clickable PGCC Campus Map built to help students learn about the campus buildings through clean pop-up panels.

# Tech Stack
HTML5 / CSS3 / JavaScript
Bootstrap 5
SVG + Fetch API
Adobe Illustrator (map cleanup + layers)
Git / GitHub

# How I did this
I exported a PNG of the most current version of the PGCC Campus map, used Adobe illustrator to re-design building number markings for interactivity
